Quiz Game Rules

Read these rules carefully before playing

Game Modes

  • Normal Mode: No time pressure. Answer at your own pace.
  • Challenge Mode: 10-second timer per question. Earn bonus points for quick answers!
  • There is different cateogry of question bank.
  • There is separate leaderboard rankings for each mode.

Game Objective

  • Complete all different cateogry.
  • Complete all 5 levels by answering questions correctly.
  • Each level has 15 questions, but only 10 random questions are shown.
  • Earn stars and points based on the performance.
  • Questions cover Different topics from basic to advanced.

Scoring System

  • Correct answer: +10 base points
  • Challenge Mode Bonus: Up to +5 bonus points for quick answers (under 10 seconds)
  • Wrong answer: 0 points (no penalty)
  • Time's Up (Challenge Mode): Question skipped, 0 points
  • 3 Stars: 95% or higher (9-10 correct answers)
  • 2 Stars: 70-94% (7-8 correct answers)
  • 1 Star: 60-69% (6 correct answers)
  • 0 Stars: Below 60% (less than 6 correct)

Level Progression

  • Level 1 is unlocked by default for all players
  • You must earn at least 1 star to unlock the next level
  • Complete Level 1 → Unlock Level 2, and so on
  • It can replay completed levels to improve your score
  • Higher levels have more difficult questions

Timer System

  • Normal Mode: Overall timer only (no per-question limit)
  • Challenge Mode: 10-second timer per question + overall timer
  • Challenge mode timer turns red when under 10 seconds remaining
  • Completion time is shown in the game header and results

Leaderboard

  • The best score for each level is saved automatically
  • Separate leaderboards for Normal Mode and Challenge Mode
  • Leaderboard ranks players by total stars earned
  • If stars are equal, total score determines ranking
  • Compete with yourself and improve your rankings!

Player Profiles

  • The progress is saved automatically when you enter a name
  • Each player name has its own profile with separate progress
  • Stars earned across all levels are tracked per profile
  • It can reset all progress with the "Reset All" button

Tips for Success

  • Read each question carefully before answering
  • In Challenge Mode, balance speed with accuracy
  • Review the correct answer when you get it wrong
  • Practice makes perfect - replay levels to improve!
  • Start with Normal Mode to learn the questions, then try Challenge Mode
